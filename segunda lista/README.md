# Calculadora de Calorias (Java)

Programa simples para calcular a quantidade total de calorias de uma refeição.

Como compilar e executar (Windows PowerShell):

```powershell
javac MealCalories.java
java MealCalories
```

Exemplo de uso:

- Digite 1 para prato vegetariano, 2 para peixe, 3 para frango, 4 para carne.
- Digite 1 para abacaxi, 2 para sorvete diet, 3 para mousse diet, 4 para mousse chocolate.
- Digite 1 para chá, 2 para suco de laranja, 3 para suco de melão, 4 para refrigerante diet.

O programa imprime o total de calorias somando prato + sobremesa + bebida.

Observação: Código feito de forma simples, no estilo de um estudante iniciante em Java.
